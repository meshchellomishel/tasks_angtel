
Задание 3.

Разработать программу, которая анализирует аргументы командной строки,
формирует на их основе JSON-документ и выводит его в терминал. Все аргументы
являются опциональными. Некоторые из них принимает одно или более значений.
Некоторые аргументы могут вводиться несколько раз. Программа принимает
неопределенное количество (от 0 до бесконечности) позиционных аргументов.

Аргументы.
1. -P/--pid - в документ добавляется PID процесса по ключу "pid". Аргумент без
значений.
2. -u/--uid - в документ добавляется UID пользователя по ключу "uid". Аргумент
без значений.
3. -g/--gid - в документ добавляется GID группы пользователя по ключу "gid".
Аргумент без значений.
4. -e/--env - в документ добавляется словарь со значениями указанных
переменных окружения. Имена переменных окружения передаются в значении --env и
являются ключами создаваемого словаря. Аргумент принимает одно обязательное
значение и может вводиться несколько раз.
5. -d/--dir - программа меняет свой текущий каталог на тот, который указан в
аргументе, и добавляет в документ список файлов в этом каталоге по ключу
"dir". Аргумент принимает одно обязательное значение.
6. -p/--pos - в документ добавляется словарь, содержащий значения указанных
позиционных аргументов. Аргумент принимает одно обязательное значение и может
вводиться несколько раз.
7. -f/--file - в документ добавляется содержимое указанного файла или
содержимое потока ввода, если указано значение "-". Аргумент принимает одно
обязательное значение.

Реализация 5, 6, 7 - задание повышенной сложности.

Пример.
Исполняется следующая команда:
$ ./lab3.py -u --gid -P -d ./my_dir -E MAIL -E USER -p 1 -p 3 -p 100 123 345 567 789

В данной строке -u, --gid, -P, -d, -E и -p являются именованными аргументами,
./my_dir, MAIL, USER, 1, 3, 100 - значениями именованных аргументов, 123, 345,
567, 789 - позиционными аргументами.

Ожидаемый вывод:
{
	"uid": 1000,
	"gid": 1000,
	"pid": 10171,
	"pos": [
		"345",
		"789"
	],
	"env": {
		"MAIL": "/var/mail/pavel",
		"USER": "pavel"
	},
	"dir": [
		"libubox",
		"lab_13",
		"Makefile",
		".lab_13.c.swo",
		"configuration.json",
	]
}
